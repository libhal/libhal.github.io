class input_pin#

Digital input pin hardware abstraction interface.

Use this to read a pin and determine if the voltage on it is HIGH or LOW.

Subclassed by hal::lpc40::input_pin, hal::mock::input_pin, hal::soft::inert_input_pin, hal::soft::input_pin_inverter, hal::stm32f1::input_pin, hal::stm32f411::input_pin
